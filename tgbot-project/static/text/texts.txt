[help]
Ты можешь выбрать одну из следующих команд.

<b>Настройки</b>
/start - Запустить бота
/help - Список команд
/welcomeconfig - Установить приветственный стикер\анимацию

<b>Аниме Команды</b>
/animesearch - Поиск аниме по ключевому слов(-у, -ам)
/animetop - Рейтинг аниме: За Все Время, Самые Популярные, Самые Ожидаемые, Лучшие Сезона, Самые Любимые
/animeost - Саундтреки аниме
/animecast - Исполнители ролей аниме
/animesummary - Синопсис аниме
/animetrailer - Трейлер аниме

<b>Избранные</b>
/watchlist - Посмотреть список Избранных
/watchlater - Добавить аниме в Избранные

<b>Дополнительные</b>
/pat - Погладить
/cheeks - Потянуть за щеки
[/help]
